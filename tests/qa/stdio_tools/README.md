# Moved: STDIO MCP tool tests

STDIO (and all other) MCP tool tests are unified under **`tests/qa/mcp_tools/`**.

Use **`tests/qa/environment.yml`** to create the `anaconda-mcp-qa` conda env, then run with `--mcp-profile=stdio-stdio` or `stdio-http`. See [`../mcp_tools/README.md`](../mcp_tools/README.md).

This folder is only a pointer for old links.
